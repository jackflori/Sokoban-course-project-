# Sokoban (15-112)
Sokoban puzzle game with some fun additions for final project in 15-112. Utilized CMU Graphics’ model-view-controller framework. 

Feature List:
- Main Menu, Win/Lose Screen
- Basic Sokoban Controls/Rules
- 3 Levels of Increasing Difficulty
- Reset Level with Enter
- Toggle Hard/Timed Mode
- Move Counter Displayed After Win
- Looped Background Music with Play/Pause Button
- Level Completion Sound Effect
 
* requires downloading cmu_graphics library which may cause some difficulties - developed solely with and runs natively on CMU CS Academy's built-in IDE *
